# Biomedical-Image-Analysis-Matlab
Projects Overview
1) Cell Segmentation: Nuclear segmentation in fluorescence microscopy using watershed &amp; morphological filtering. 
2) Image Registration: Affine transformation and alignment quality assessment using Mutual Information.  3) 3.D Computer Vision: STL file processing and perspective projection modeling.
4) Deep Learning & Image Classification: Comparative analysis of AlexNet and VGG16 architectures using Transfer Learning on the CALTECH101 dataset. Achieved 86.81% accuracy with VGG16, including performance evaluation via confusion matrices and loss curves.
